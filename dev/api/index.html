API

SimUtils.Correlation — Method
Correlation(a[, b=a]; subtract_mean=false)

Linear (non-circular) correlation via FFT with zero padding and normalization by the decreasing sample count (N:-1:1). If b==a, this is the autocorrelation.

source
SimUtils.Correlation_GPU — Method
Correlation_GPU(a[, b=a]; subtract_mean=false)

GPU version of Correlation. Returns result on CPU.

source
SimUtils.FT — Method
FT(t, x; tol=1e-8, indvar=true)

Discrete FT returning angular frequencies ω and spectrum X(ω) on a uniformly spaced time grid t. Scaling is such that X = dt * FFT(x) (up to fftshift), with an origin correction so you can use iFT(...; t0=t[1]) to invert exactly.

source
SimUtils.FT_GPU — Method
FT_GPU(t, x; tol=1e-8, indvar=true)

GPU version of FT. Only the FFT itself runs on the GPU; results are returned on the CPU.

source
SimUtils.MSD — Method
MSD(positions)

Mean-squared displacement computed from autocorrelation: MSD(τ) = 2 (C(0) - C(τ)), where C is the autocorrelation of the position time series.

source
SimUtils.MSD_GPU — Method
MSD_GPU(positions)

GPU-backed autocorrelation; returns MSD on CPU.

SimUtils.iFT — Method
iFT(ω, X; tol=1e-8, indvar=true, t0=0.0)

Inverse of FT. If you originally used FT(t,x) with time origin t[1], call as iFT(ω, X; t0=t[1]) to recover the original grid. Returns (t, x) with t = t0 .+ (0:N-1)*dt, where dt = 2π/(N Δω).

source

GPU version of iFT. Returns arrays on the CPU.

source
SimUtils.pad_to_power_of_two — Method
pad_to_power_of_two(x, meanval)

Return a copy of x zero-padded to the next power-of-two length after first subtracting meanval from the populated portion.

This helper is used internally to make FFT-based correlation routines more efficient while keeping the linear-correlation normalization explicit.
